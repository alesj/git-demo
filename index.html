GCloud Java: Idiomatic Java Client for Google Cloud Platform services.

This is a Java Client for accessing Google Cloud Platorm services such as Datastore, Storage, PubSub and others. This library is in a early stage of its development and may occasionally make backwards-incompatible changes, but it is already usable.
